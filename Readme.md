# Blazor Start

This is a rewrite of my [New Tab Page project](https://github.com/NeilBrommer/NewTabPage) using
[Blazor](https://dotnet.microsoft.com/apps/aspnet/web-apps/blazor).

## Structure

- Bookmark containers (tabs along the top)
- Bookmark groups
- Bookmarks
